## 📊 Module Quick Reference

### Module 1: FVG Engine
**Fair Value Gaps — Automatic gap detection**
- Detects 3-candle gaps (bullish & bearish)
- Color: Green (bullish) / Red (bearish)
- Settings: Opacity, width, max cap
- Status: ✅ Production-ready
- See: **README.md** Module 1 / **TESTING_GUIDE.md** Suite 1

### Module 2: SMT Divergence
**Synergistic Market Timing — Pair correlation**
- Works on: NQ1!/ES1! and MNQ1!/MES1!
- Silent on: Forex/crypto (expected)
- Visual: Dashed lines + "SMT ↑/↓" labels
- Status: ✅ Production-ready
- See: **README.md** Module 2 / **TESTING_GUIDE.md** Suite 2

### Module 3: 50% Daily Range
**Session Midpoint Tracker — Equilibrium level**
- Resets: 18:00 NY (UTC-4) daily
- Visual: Horizontal line + "EQ {price}" label
- Updates: Live as range expands
- Status: ✅ Production-ready
- See: **README.md** Module 3 / **TESTING_GUIDE.md** Suite 3

### Module 4: Session Killzones
**Trading Zone Highlighting — 4 sessions**
- Sessions: Asia, London, NY Open, NY Close
- Visual: Colored background during active hours
- Customizable: All times editable
- Status: ✅ Production-ready
- See: **README.md** Module 4 / **TESTING_GUIDE.md** Suite 4

### Module 5: Swing Points + Equal Levels
**Multi-Timeframe Pivots — With sweep detection**
- Timeframes: 1H, 4H, Daily
- Sweep countdown: 5/4/3 candles per TF
- Equal levels: Within 15 ticks (customizable)
- Status: ✅ Production-ready
- See: **README.md** Module 5 / **TESTING_GUIDE.md** Suites 5–6
